# Tindev - Frontend

## Features
- Aplicação frontend do Tindev WEB

## Tecnologias
- React

## Dependências
- Axios
- Dotenv
- socket.io-client

## Variáveis de configuração
Crie um arquivo `.env` na raiz do seu projeto.

`
REACT_APP_API_URL=https://suaapi.com
`

## Como configurar e executar

```bash
$ npm install

$ npm start
```
